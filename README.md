# HVAC Service Platform

Full-stack project:
- Frontend: React (CRA) + Redux
- Backend: Django REST Framework (JWT)

## Quick start

### Backend
1. Create/activate a Python virtualenv
2. Install requirements: `pip install -r backend/requirements.txt`
3. Run: `python backend/manage.py migrate`
4. Run: `python backend/manage.py runserver`

### Frontend
1. Install deps: `cd frontend && npm install`
2. Run: `npm start`

Frontend: http://localhost:3000
Backend API: http://localhost:8000/api/v1/
